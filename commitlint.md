## Allowed commit types

- feat: new feature
- feat(auth): add refresh token

- fix: bug fix
- fix(auth): handle expired token

- docs: documentation only
- docs(readme): update setup steps

- style: formatting only (no behavior change)
- style(ui): format templates

- refactor: refactor (no feature, no fix)
- refactor(auth): simplify guards

- erf: performance improvement
- perf(core): reduce change detection work

- test: add/fix tests
- test(auth): add login spec

- build: build system / deps
- build: bump angular to 19

- ci: CI config/scripts
- ci(github): add lint job

- chore: maintenance tasks
- chore: update tooling

- revert: revert a commit
- revert: feat(auth): add refresh token
