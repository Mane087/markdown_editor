- Italic `ctrl+alt+i`
- Underlined `ctrl+alt+s`
- Quote `ctrl+alt+q`\
- Lowercase `ctrl+alt+a`
- Uppercase `ctrl+alt+u`
- Headings H1 - H6 `ctrl+1/2/3/4/5/6`
- List `ctrl+alt+l`
- Lista Numeric `ctrl+alt+n`
- Horizontal Rule `ctrl+alt+r`
- Code inline (highlighted) `ctrl+alt+c`
- List of Alerts
- Download .md file
- Upload .md file
- Hide/Shoe preview `ctrl+alt+p`
- Full/Min preview `ctrl+alt+f`
- Global search
