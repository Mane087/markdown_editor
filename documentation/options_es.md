# Opciones

- Negritas `ctrl+alt+b`
- Cursiva `ctrl+alt+i`
- Sub-rayado `ctrl+alt+s`
- Cita `ctrl+alt+q`
- Transformar a minusculas `ctrl+alt+a`
- Transformar a mayusculas `ctrl+alt+u`
- Titulos H1 - H6 `ctrl+1/2/3/4/5/6`
- Lista `ctrl+alt+l`
- Lista Ordenada `ctrl+alt+n`
- Division `ctrl+alt+r`
- Linea de codigo (resaltado) `ctrl+alt+c`
- Lista de Alertas
- Descargar archivo .md
- Mostrar/Ocultar vista previa `ctrl+alt+p`
- Agrandar/Minimizar vista previa `ctrl+alt+f`
